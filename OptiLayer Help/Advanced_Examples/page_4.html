Foreword

In order to work with advanced examples you will need special examples installed at your hard disk. The names of all these directories start with the prefix LEC.

If you use retail version of OptiLayer, make sure that you selected “Custom” setup type during the installation, and that “Examples” topic at the next page of the setup procedure was selected.

If you use DEMO version of OptiLayer, example directories are installed automatically. Please, remember, that all files are read-only in DEMO version, so data at the hard disk can not be modified.

Any modern computer running Windows 10, Windows 8.1, Windows 7, or Windows Vista can be used to work with example directories. Some of advanced examples are computationally extensive. See Hardware Requirements section for more detail.

Descriptions of the advanced examples are brief and highlight only the most essential features. It is recommended to pass first Tutorial exercises in order to get initial acquaintance with OptiLayer.

Important Note: Please, don’t use Tutorial and Example directories for your own data. If you reinstall OptiLayer Thin Film Software, Tutorial and Example directories will be overwritten and any new data stored there will be lost.
