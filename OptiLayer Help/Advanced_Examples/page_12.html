 Note: Due to the continuous improvement of OptiLayer design options and random nature of some of their elements, the results obtained may differ from those previously saved in the design database.

The main purpose of this example is to demonstrate that for coatings designed to perform under oblique light incidence, the use of a third layer material with an intermediate refractive index can be critical to design success.

In the File -> Project menu of OptiLayer, open the directory LEC24D8. The datafiles needed to run this exercise are already loaded into RAM. These are the substrate Glass n=1.52, layer materials n=2.35 and n=1.46, the beamsplitter spectral target BS 500-600 nm. The target sets the demand for 50% transmittance for the s- and p-polarized light at 45-degree light incidence. Note that we consider a beamsplitter immersed in glass. This is specified by loading the substrate refractive index Glass n=1.52 as Incident Medium (see the General Information window).



First, we will try to design a beamsplitter using only two materials with high and low refractive indices. In the General Information (GI) window, check that only two materials are loaded (If there is a third material with a refractive index of 1.63, select it and use the Remove command at the right mouse click). Also check that there is no design in the GI window. If any design still exists, remove it as we will run the gradual evolution design procedure without a starting design. To remove a design shown in the GI window, click its name and select the empty row at the top of the selection list.

Select the Gradual Evolution … mode in the Synthesis menu. In the Setup window, activate only one termination criterion, Max optical thickness with a value of 15000 nm.

Make sure that Sequential quadratic programming (SQP) is selected as the local optimization method. This can be checked either in the GI window or later after pressing the OK button in the Setup window and opening the Gradual evolution window. To see the selected method, choose Options at the bottom of this window. If necessary, the method can be changed in the Analysis and Synthesis Options window. Note that the local optimization method can also be changed directly in the GI window by clicking the method name in the Refinement line of the GI window.

Before starting the gradual evolution procedure, clear the History part of the window using the Clear button at the lower right corner of the window. After this, start the design procedure by clicking the Run button in the lower left corner of the window. At a given maximum level of the design total optical thickness, a satisfactory result will not be achieved.

If you try to increase the Max optical thickness value, there will be only a slight improvement in the obtained MF values. Unfortunately, in the design problem under consideration, satisfactory results cannot be achieved using only two materials with high and low refractive indices.

Load the third material with a refractive index of 1.63. Make sure there is still no design in the Design line of the GI window. If by chance one of the obtained designs was loaded into the GI window, remove it as described above. Again, we will run the gradual evolution procedure without a starting design. Clear the History part and click the Run button. Monitor the progress of the procedure on the History part of the window. With total optical thickness (TOT) values of around 5000 nm or more, much better results are achieved than before.

Stop the procedure at TOT values exceeding 9000 nm as excellent results have already been achieved. The rms deviations of s- and p-transmittances from 50% are within 0.01% and further improvement of MF by complicating the design does not make sense.

We see that a dramatic improvement in MF values was made possible by using a third layer material with an intermediate refractive index.
