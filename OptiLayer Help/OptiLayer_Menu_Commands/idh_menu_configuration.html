Export Mode and Simple Mode selection allow an easy switch between different levels of the user interface.

Multiprocessing - this option allows the use of parallel computations on several processors or several processor cores. It is possible to fine-tune CPU utilization by specifying a custom number of cores. This might be useful if another program needs to run computations together with OptiLayer. Leaving some number of cores unoccupied by OptiLayer will help to avoid the so-called over-subscription problem.
