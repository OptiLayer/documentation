Formula Constrained Optimization

Formula constrained optimization is a specialized optimization option based on the formula representation of a multilayer coating. Some formula parameters may be considered as unknown parameters, some of which are continuous variables changing within certain limits, while others may be integer parameters with a predefined set of values. For example, a quarter-wave stack with n pairs of layers, along with another non-quarter wave stack with m pairs and QWOT thicknesses equal to a and b, can be represented as:

 (H L)^n (a H b L)^m

We can consider a set of possible values for integers n and m. For each combination of n and m, OptiLayer will attempt to find the optimal values of continuous parameters a and b, optimizing the design performance with respect to the loaded targets. Another example could be a stack with periods having varying optical thicknesses. Ultra-wide range high reflectors, chirp mirrors, and other coatings can be designed using this method. Consider a formula:

 ((a*iL+b)H (c*iL+d)L)^n

where a, b, c, and d are continuous variables, and iL is a predefined integer equal to the layer number in the design. Additional constraints are implicitly formulated for expressions in this formula:

 (a*iL+b) ≥ 0

and

 (c*iL+d) ≥ 0

More complex problems can be specified using the Formula Constrained Optimization option.

 Note 1: Layers in the formula in this option are numbered starting from the substrate.

 Note 2: Two-character abbreviations are allowed, so it is necessary to separate coefficients and material abbreviation symbols by blanks.

 Note 3: To differentiate between variables and material abbreviations, it is recommended to use small letters for variables and capital letters for material abbreviations.
