Random Optimization

` <inhomogeneous_refinement_setup.html>`__ ` <idh_menu_synthesis.html>`__ ` <idh_random_optimization_parame.html>`__

This procedure can be useful for solving complex design problems when good starting designs are unknown. Another application is to use this option to generate a set of different designs for finding a design with better manufacturability, for example:

  • A. V. Tikhonravov and M. K. Trubetskov, “Design Opportunities for Better Manufacturability,” in Optical Interference Coatings, OSA Technical Digest (CD) (Optical Society of America, 2007), paper WA2. http://www.opticsinfobase.org/abstract.cfm?URI=OIC-2007-WA2

  • A. V. Tikhonravov and M. K. Trubetskov, “Modern status and prospects of the development of methods of designing multilayer optical coatings,” J. Opt. Technol. 74, 845-850 (2007) http://www.opticsinfobase.org/abstract.cfm?URI=JOT-74-12-845

The program will prompt you to specify Random Optimization Parameters (the number of layers in a random starting design, the average value of physical layer thicknesses, and abbreviations specifying the materials for odd and even layers). Alternatively, Random Optimization can search in the vicinity of the currently loaded design, providing a more detailed search. The program generates a sequence of starting designs with the specified number of layers and average layer thickness. Each design is optimized using the method selected in the Synthesis Options. The process continues until it is interrupted by the user, the best results being stored in the Collection storage. This process continues until interrupted by the user, with the best results stored in the Collection storage. These results can later be used as starting designs for further optimization. They can be retrieved from the Collection storage and loaded into memory using the Extract command. If a target function, substrate material, and/or layer materials have not been loaded, you will be prompted to do so. If fewer than two layer materials have been loaded, the random optimization procedure cannot be run. You can monitor the progress of the Random Optimization procedure in the Refinement Window.
