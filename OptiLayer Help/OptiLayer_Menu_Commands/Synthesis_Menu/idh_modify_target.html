Modify Targets

` <modify_stack.html>`__ ` <idh_menu_synthesis.html>`__ ` <modify_integral_target.html>`__

This option allows you to modify a targets function residing in memory. The user can choose which type of target they want to modify.

When you start the Modify Targets command from the Synthesis menu, you can modify only loaded Target. It is often useful to modify a target function during the optimization process. You can restore the initial target function at any time by reloading it from the Target database. To make the changes to the edited target function permanent, you should save it in a Target database with the help of the Save target command from the Data menu.

The Apply button allows you to apply modifications without closing the window, while the OK button applies modifications and closes the window.
