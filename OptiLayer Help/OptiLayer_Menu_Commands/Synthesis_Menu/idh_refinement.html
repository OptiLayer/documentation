Refinement

Refinement is performed by optimizing the merit function with respect to the layer thicknesses. The merit function is constructed in accordance with the Target function and Target Qualifiers.

You can select the optimization method that best suits your problem. New optimization methods used by OptiLayer provide rapid convergence of the optimization procedure.

OptiLayer provides a set of optimization methods for the user to choose from:

  • Hyper Newton method,

  • Modified damped least squares (Modified DLS),

  • Newton’s 2nd order method,

  • Quasi-Newton DLS

  • Sequential Quadratic Programming (SQP) method,

  • Conjugate gradient method,

  • Steepest descent method.

For information on these methods, refer to the optimization methods.

Compared to other methods, Newton’s method and the Hyper Newton method require more computations at each stage of the refinement process, but this is often justified by more rapid convergence. Newton’s method is particularly effective when applied to slowly converging problems or when first-order optimization methods fail. The user can visually control the refinement process in the Refinement Window.
