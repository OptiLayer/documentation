Configuration Options allow the user to set some of the most general settings that are used throughout the program. They include the following items:

  • Mode  - allows the user to switch between the data representations in Spectral or Angular modes.

  • Units - allows the user to select units for measuring lengths, frequencies, etc.

  • Thickness Units - allows the user to switch between physical, optical, quarter-wave, or full-wave representations of layer thicknesses.

  • Source/Detector/Cone/Integral - allows the user to set how light sources and detectors are used during computations of T, R, I, and units of spectral distribution.

  • Color - allows the user to set various parameters affecting the color evaluation module. These parameters include Reference White, Observer, and Integration step. A custom Reference White light source can also be specified as a blackbody or a file from the Light Source/Detector database.

  • GD & GDD - allows the user to select units for Group Delay, Group Delay Dispersion, and Third Derivative Dispersion.

  • WDM - allows the user to specify Transmission level and Rejection level for the WDM Filter synthesis option. The user can also specify whether it is necessary to set half-width requirements at the 50% transmittance level.

  • Data - allows the user to adjust the look of the Database dialog and to set miscellaneous options including sorting order for database files by Name, Author, Creation Date, or Revision Date.

  • Time/Date - enables or disables the date and time display in the OptiLayer window and selects a style of time display.

The Multiprocessing option allows the activation of simultaneous use of several processors (or processors’ cores) for parallel computations.
