Setting User Name

It is convenient to distinguish between data created by different users.

Enter a name in the User Name field to identify the database files you create. You may enter any combination of up to eight letters; the best choice may be your own name. If you don’t want to be prompted for a User Name the next time you start OptiLayer, check the box “Display this prompt next time.” If necessary, you can change the user name and/or restore the display of this dialog in Options/Data Dialog, started from the Configuration Menu.
