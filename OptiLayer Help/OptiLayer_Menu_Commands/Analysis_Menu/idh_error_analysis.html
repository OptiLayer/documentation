Errors Analysis

The Error Analysis command in the / Analysis menu initiates a computational procedure that provides a statistical evaluation of the effect that errors in the layer thicknesses and refractive indices in a design will have on the spectral response of a designated spectral characteristic. As the computations proceed, a sequence of curves corresponding to the statistical tests being performed is plotted on the screen. During the computations, the progress of the statistical procedure is indicated by the percentage bar at the bottom of the window. This command of the / OptiLayer Analysis tools allows estimating the influence of errors in layer thicknesses and refractive indices on the spectral characteristics of the current design. The Error Analysis computation can be performed for only one spectral characteristic at a time. You must specify the characteristic (Transmittance, Reflectance, etc.), the state of polarization, and the angle of incidence prior to the calculations (see Setting Error Analysis Characteristics).

You can perform Error Analysis for layer thickness errors, refractive index errors, or both. In all cases, it is necessary to specify the root-mean-square errors of the corresponding parameters in special setup windows. Since the estimation of error influence is a statistical procedure, a reasonable number of tests should be specified (by default, the number of tests is 20). The width of the probability corridor corresponds to the magnitude of the errors that were set in the Error Analysis Setup. The default setting for the width of the probability corridor is for one standard deviation. For more detail on the error evaluation output, see / Error Analysis Results.
