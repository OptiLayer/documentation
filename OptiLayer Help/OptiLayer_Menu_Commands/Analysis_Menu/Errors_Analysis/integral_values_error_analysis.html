Integral Values Error Analysis

` <electric_field_error_analysis.html>`__ ` <idh_error_analysis.html>`__ ` <error_yield_analysis.html>`__

If Integral Values Error Analysis is selected in the Error Analysis Characteristics dialog, then an additional Error Analysis of Integral Values window is displayed.

In this window, the theoretical value is represented along with its mathematical expectation and the corridor of errors.

Note that the Integrals, Values window should be configured before running this option. The list of values subjected to error analysis is selected according to the configuration of the Integrals, Values window.
