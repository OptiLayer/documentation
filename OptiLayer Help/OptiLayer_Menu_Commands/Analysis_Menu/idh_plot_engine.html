Plot Engine

` <creatematerialassistant.html>`__ ` <idh_menu_analysis.html>`__ ` <idh_plot_engine_1.html>`__

OptiLayer provides a universal tool for creating presentation-quality 2D and 3D plots of various spectral characteristics. It is accessed via the Plot Engine command in the / Analysis menu. The Plot Engine plots the characteristics of the design currently in memory.

The procedure for setting up parameters of the graphic window and characteristics to be plotted consists of three consecutive steps (represented by the different pages of the Data Plot Wizard window):

  • Selecting a plotting mode (2D or 3D) and characteristics to be plotted: To Plot page;

  • Setting up the page layout and labeling (legend, header, axes names): Layout page;

  • Choosing the window to display the plot in (either in the existing window or in a new window): Destination page.
