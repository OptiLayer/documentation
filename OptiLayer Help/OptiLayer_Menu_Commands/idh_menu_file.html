File Menu

The File submenu provides access to various project directories stored on the hard disk, enabling the transfer of data files between different project directories, as well as the option to backup and restore these directories.

  • Problem Allows the user to choose a Problem Directory.

  • Recent Problems Allows for quick selection of previously accessed project directories.

  • File Transfer Opens a submenu for initiating the File Transfer utility to exchange data between various project directories.

  • Backup/Restore Opens a dialog box to create a backup copy of your project directory or to restore previously saved data (refer to Problem Directory Backup).

  • Catalog Allows loading of a substrate from a Catalog of known substrates. The catalog currently includes Schott, Hoya, Ohara, Ohara Low Tg, Metals, CDGM, Sumita, Hilkari and other data files. Information on Layer Materials, Light Source and Detector catalogs is also accessible.

  • Refractiveindex.INFO Opens a database that can also be accessed on the Internet at http://refractiveindex.info. Users can select information on refractive indices and extinction coefficients of various materials and import them into OptiLayer for further use.

  • Open Layout Opens a previously saved OptiLayer Layout file (.olprj extension) and substitutes all settings related to loaded data, configuration, and window layout with the data stored in the OptiLayer Layout file. This option is also accessible by pressing Ctrl+O.

  • Save Layout As Allows the user to save an OptiLayer Layout file with the extension .olprj for future use.

  • Save Chart / Bitmap / Output As Opens a standard File Save dialog box to choose the name of an output file. If a text output of any data is produced and the text output window is active, an ASCII file with the default extension .out is generated (refer to Creating Text Output). Otherwise, the current window is saved as an image in one of the graphic formats (.bmp, .jpg, .wmf, .gif, .png, .pcx, .tiff, etc.). If a chart was selected prior to this action, an additional .vtc2 format supported by OptiLayer Plot Engine will also be accessible.

  • Record Animation… Enables the recording of animated .gif files frame by frame or automatically during ongoing computations. These Animations can be played in any internet browser or inserted into presentations.

Import

  • TFCalc Data File Allows TFCalc users to import their data to OptiLayer quickly. Coatings files (*.tfd) are imported with any supplementary files required. In order to support automatic import, OptiLayer should know the location of the TFCalc executable file. It is also possible to import Material files (*.mat) and Illuminator/Detector files (*.ill;*.det).

  • Essential Macleod Data File Allows Essential Macleod users to import their data to OptiLayer quickly. Design files (*.dds) and Stack files (*.dst) are imported with any supplementary files required. In order to support automatic import, OptiLayer should know the location of the Essential Macleod Materials directory (usually “c:\Program Files\Thin Film Center\Materials\”). Select this path at the bottom part of the Open Essential Macleod file dialog. It is also possible to import Layer Material and Substrate files (*.tfp) directly.

  • LZH Design Allows the import of Laser Zentrum Hannover SPEKTRUM file format (.dsg). This format is used by on-line deposition software, including Broad-Band Monitoring systems.

  • Sopra n,k Data Allows the import of Sopra dispersion data directly to OptiLayer. Currently, the Sopra database can be freely downloaded from http://www.sopra-sa.com. Files can be imported either to the Substrate/Incident Medium database or Layer Material database. The type of database is determined by the proper selection of the Files of Type field in the Import Sopra n,k Data dialog.

  • ZEMAX Glass Data Allows the import of dispersion data for a material or substrate represented in ZEMAX Glass format (.agf). After opening the .agf file, the Import ZEMAX Glass data dialog will allow selection of materials to be imported.

  • Bühler Leybold Dispersion Data Allows the import of dispersion data describing Substrate, Incident Medium, and Layer Materials stored in Bühler Leybold format (*.ldd files).

  • Bühler Leybold Monitoring Report Allows the import of coating design, layer materials, substrate, and monitoring spreadsheet stored in Bühler Leybold Monitoring Report format (*.lmr files).

Export

  • Export to Excel… (F11) Enables the user to export any spreadsheet or report (found in the Results menu) to Microsoft Excel. ** **

  • Export Chart Opens the Export Chart dialog that allows the user to save the currently selected chart in various graphical or data formats.

  • Bühler Leybold Monitoring Report Enables the user to export the Monitoring Spreadsheet to Bühler Leybold software for controlling deposition plants. A Monitoring Spreadsheet must be created beforehand for this export.

  • LZH Design Export the current design, substrate, and layer materials in Laser Zentrum Hannover; SPEKTRUM format (*.dsg).

  • ZEMAX Coating (Ctrl+Alt+Z) Enables the user to export the current design and layer materials to the ZEMAX Coating file or ZEMAX performance file.

  • CODE V Sequence (Ctrl+Alt+V) Export the current coating design in CODE V Sequence file format.

Other

  • Page Setup Opens the Page Setup dialog.

  • Print Preview Opens the Print Preview dialog, allowing the user to view and adjust the printer output. Print Preview is available for Windows with plots only.

  • Print Allows the user to print a text output rather than saving it to a file. This option is available when generating a text output (see “Creating Text Output”).

  • Exit Exits the program. This can also be done by pressing Alt+X.
