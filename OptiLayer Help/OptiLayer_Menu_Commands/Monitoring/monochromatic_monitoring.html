Monochromatic Monitoring Simulation

` <broadmonsim_res.html>`__ ` <idh_menu_analysis.html>`__ ` <single_dep_rates.html>`__

The Monochromatic Monitoring Simulation option is another OptiLayer tool that is designed for conducting experiments on the computational manufacturing of optical coatings.

The Monochromatic Monitoring Simulation option allows the user to conduct computational experiments simulating deposition processes in vacuum chambers equipped with monochromatic monitoring devices. A five-step dialog is used to specify parameters of the deposition process and monitoring device, run computational experiments, and analyze their results.
