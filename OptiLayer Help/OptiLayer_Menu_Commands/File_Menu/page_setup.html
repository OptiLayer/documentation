Page Setup

The Page Setup dialog in OptiLayer allows you to control various aspects of the printed output, including:

  2. Paper size: You can choose the size of the paper on which the output will be printed, such as A4, Letter, Legal, etc.

  3. Margins: Adjust the margins of the printed page to control the spacing around the edges of the output.

  4. Header/Footer: Customize the header and footer content that will appear on each printed page. This can include page numbers, document title, date, and other information.

  5. Scaling: Control the scaling of the printed output to fit within the specified paper size. You can adjust the scaling to ensure that the content fits correctly on the printed page.


By using the Page Setup dialog, you can tailor the printing settings to meet your specific requirements and preferences for the printed output in OptiLayer.
