Layer Optimization Status

` <idh_design_formula.htm>`__ ` <idh_design_editor.htm>`__ ` <design_inhom_inter.htm>`__

To manage how a layer is treated during the refinement or synthesis process, it receives an optimization status. There are two potential statuses a layer can be given:

  • Active (A)

  • Fixed (F)

By default, every layer is assigned the status of Active. During the refinement or synthesis process:

  • The thickness of an Active layer can be altered. Moreover, any material that is loaded into the system and defined as either Active or available for substitution (referenced in the document as Exclusionary under Layer Material Status) can be incorporated into it.

  • The thickness of a Fixed layer cannot be modified, nor can any other materials be introduced or substituted into it.

The optimization status of a layer within a currently loaded design can be altered via the Modify Design option found within the Synthesis menu. To adjust the optimization statuses of layers within designs stored in the Design Database, you should access the Design Database, select the design of interest, and then employ the “Edit” function as you normally would.

On rare occasions, the optimization status of a Fixed layer can change to Active during the synthesis or refinement process. This happens because OptiLayer merges adjacent layers of the same material into a single layer. If a Fixed layer is merged with an Active layer, the resulting layer assumes an Active status. This situation can occur, for example, when a thin layer of a different material is removed during the refinement process, potentially leading to unpredictable outcomes with respect to the original design intent. One solution to prevent unintended merging and status changes is to create an “alias material.” This involves defining a material with the same optical properties as the material of the Fixed layer, but under a different name and abbreviation. Since OptiLayer does not merge two layers with different names, this strategy effectively resolves the issue.
