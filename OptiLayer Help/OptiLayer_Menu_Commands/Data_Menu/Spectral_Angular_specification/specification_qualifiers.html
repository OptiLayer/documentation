Specification Qualifiers

In defining specifications, qualifiers play a crucial role. You can assign a qualifier (A, B, R) to any spectral/angular point in the specification.

  • A corresponds to the “Bypass Above” option, where OptiLayer will bypass the target if the corresponding characteristic exceeds a specified target value.

  • B corresponds to the “Bypass Below” option, indicating that OptiLayer will bypass the target if the corresponding characteristic is below a specified target value.

  • R corresponds to the “Range” option, where OptiLayer will bypass the target if the corresponding characteristic falls within a specified segment. When using the Range qualifier, an additional column is automatically added in the Specification editor to specify upper values for the ranges.

For setting Specification qualifiers, you can refer to the Spectral/Angular Specifications or Color Specifications within the OptiLayer software.
