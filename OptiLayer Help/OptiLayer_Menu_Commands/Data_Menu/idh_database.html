Layer Material Database

` <table_simplify.html>`__ ` <idh_menu_data.html>`__ ` <idh_layer_material_status.html>`__

The Layer Material Database is used to store data on layer materials. To use layer materials in calculations, the user must first load them into the computer memory (see Loading Data for details).

Materials also need to be linked to the design layers, which is accomplished using two-letter material abbreviations assigned to each layer. When the materials are loaded, they are automatically assigned a single-letter character as an abbreviation. To access a database file, open the corresponding database access window by selecting the appropriate item from the Data menu or the OptiLayer Toolbar.

To edit materials properties, see Editing Material Properties
