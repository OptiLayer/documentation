Target Qualifiers

Design targets (and consequently the merit function) can be adjusted using qualifiers. You can assign a qualifier (A, B, or none) to any target.

  • A corresponds to the “Bypass Above” option. This means that OptiLayer skips the target when the corresponding characteristic exceeds a certain target value.

  • B corresponds to the “Bypass Below” option. This means that OptiLayer skips the target when the corresponding characteristic falls below a target value.

  • R corresponds to the “Range” option. This indicates that OptiLayer skips the target when the corresponding characteristic falls within a specified segment. When using the Range qualifier, an additional column automatically appears in the Target editor to specify upper values for the ranges.

  • An empty qualifier (blank) corresponds to a standard target.

To assign target qualifiers, please refer to the Editing Target or Color Target guides.
