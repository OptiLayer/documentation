Spectral Characteristics Abbreviations

` <idh_target_qualifier.html>`__ ` <target_editor.html>`__ ` <idh_spectral_characteristics.html>`__

In OptiLayer, you can specify a spectral characteristic using the following character abbreviations:

  • R: Reflectance

  • BR: Back Reflectance

  • T: Transmittance

  • A: Absorptance

  • PR: Phase shift on reflection

  • PBR: Phase shift on back reflection

  • DPR: Difference between phase shifts on reflection for s- and p-polarized waves

  • DBPR: Difference between phase shifts on back reflection for s- and p-polarized waves

  • PT: Phase shift on transmission

  • PDT: Phase difference on transmission (the difference between a wave propagated through the coating and the wave propagated the same distance in the Incidence medium)

  • DPT: Difference between phase shifts on transmission for s- and p-polarized waves

  • GDR: Group delay on reflection

  • GDDR: Group delay dispersion on reflection

  • GDT: Group delay on transmission

  • GDDT: Group delay dispersion on transmission

  • GDBR: Group delay on back reflection

  • GDDBR: Group delay dispersion on back reflection

All these abbreviations (except for DPR and DPT) should be accompanied by the following Polarization state abbreviations:

  • S, P: for completely polarized light

  • A: for unpolarized light.

Note: The angle of incidence is always specified in the Incident Medium. For back reflectance and similar characteristics, this means that the angle of back reflectance should be computed using Snell’s Law if the Exit Medium is different from the Incident one.
