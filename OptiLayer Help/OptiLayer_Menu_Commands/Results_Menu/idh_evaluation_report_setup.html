Evaluation Report Setup

` <exporttomsexceloptions.html>`__ ` <idh_menu_results.html>`__ ` <gdandgddreport.html>`__

Use the Evaluation Report Setup window to specify a spectral (spectral mode) or angular (angular mode) grid for the Evaluation Report.

In the Spectral mode, you can set a wavelength range (Wavelength min, Wavelength max), a step of the wavelength grid, and a number of spectral points for the Evaluation Report. When you change one of these parameters, other parameters are kept in correspondence automatically. Press OK to run the evaluation procedure and to display the Evaluation Report.

In the Angular Mode, you are prompted to set the range of incidence angles, the angular step, and the number of angular points.
