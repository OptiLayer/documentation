Layer Absorptance Report Setup

` <idh_ellipsometry_report_setup.html>`__ ` <idh_menu_results.html>`__ ` <material_dispersion_setup.html>`__

Use the Layer Absorptance Evaluation Report Setup window to specify the spectral (Spectral mode) or angular (Angular mode) grid for the Layer Absorptance Evaluation Report.

In the Spectral mode, you can set a wavelength range (Wavelength min, Wavelength max), a step of the wavelength grid, and a number of spectral points for the Layer Absorptance Evaluation Report. When you change one of these parameters, the other parameters are adjusted automatically. Press OK to run the Layer Absorptance Evaluation procedure and display the Layer Absorptance Evaluation Report.

In the Angular Mode, you are prompted to set the range of incidence angles, the angular step, and the number of angular points.
