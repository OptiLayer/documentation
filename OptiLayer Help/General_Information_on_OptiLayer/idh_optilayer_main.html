OptiLayer Menu and Toolbar


When OptiLayer starts, the main menu and toolbar are displayed. Most of the menu entries open pop-up sub-menus. The most frequently used commands are also accessible from the toolbar. To access different functionalities within OptiLayer, you can select an item from the following list:

  • File: Contains options for opening, saving, and managing files.

  • Edit: Includes functions for editing the database or spreadsheet.

  • View: Allows customization of the display and view settings.

  • Data: Provides tools for managing and analyzing data.

  • Analysis: Access various analytical tools and features.

  • Monitoring:

  • Synthesis: Options for synthesizing and designing optical coatings.

  • Results: Displays results and analysis outcomes.

  • Configuration: Customize and configure OptiLayer settings.

  • Window: Manage and arrange open windows and views.

  • Help: Access the help documentation and support resources.

You can click on the corresponding menu option or use the toolbar buttons to access the desired functionality within OptiLayer. Each menu entry and toolbar button corresponds to specific actions or features to help you efficiently navigate and utilize the software.
