Results obtained by OptiLayer can be presented in various reports such as design report, evaluation report, errors report, layer material report, etc. All reports may have hierarchical form, editable form, or plain form, which allows meeting even the most sophisticated user requirements. It is possible to select any report area and copy it to the Windows clipboard. The user can save and print any report or any OptiLayer window. Windows can be saved as:

  • *.bmp files (bitmap graphics format),

  • *.jpg files (JPEG bitmap),

  • *.wmf files (Windows metafiles format),

  • *.pcx files (Paintbrush format),

  • *.gif files (CompuServe bitmap),

  • *.png files (Portable Networks Graphics),

  • *.tiff files (Tagged Image File Format).

Chart Windows (windows containing plots and diagrams) have additional export formats that can be accessed through the Export Chart… command or by using the right-click popup menu:

  • *.vtc2 files (this format is used by the powerful OptiLayer Plot Engine II tool),

  • *.pdf files (these files can be opened with Adobe Acrobat tools, http://www.adobe.com)

  • *.eps PostScript files (encapsulated postscript version, these files are suitable for many printers),

  • *.svg files (Scalable Vector Graphics, http://www.w3.org/Graphics/SVG),

  • *.vml files (Vector Markup Language), http://www.w3.org/TR/NOTE-VML,

  • *.xaml files (Extensible Application Markup Language), defined at http://en.wikipedia.org/wiki/Extensible_Application_Markup_Language

Note: OLE (Object Linking and Embedding) is supported by Plot Engine II.

OptiLayer allows the user to simultaneously save all data files presented in the RAM and related information in a special OptiLayer Design Specification format (ODS format). This format is easy to read and edit, providing an additional option for data exchange and storage. OptiLayer also has the capability to export reports directly to MS Excel in one mouse click.

Users can create text outputs of database files as well as text reports of all evaluation and synthesis results.

  • To generate a text report on the database file content, click the Output button in the Database Window menubar.

  • To generate a text report on the design resident in memory, use the Current Design Report command from the Results menu.

  • To generate a text report on all designs stored in OptiLayer History or Collection files, use the History Report or Collection Report commands from the Results menu.

  • To generate text reports on the results of the evaluation modes T & R & Phase, GD & GDD, and Ellipsometric Angles, select the corresponding commands (Evaluation Report, GD and GDD Report, Ellipsometry Report) from the Results menu.

If Error Analysis or Worst Case Analysis was performed, a text report on the corresponding results can be obtained using the commands Errors Report or Worst Case Report from the Results Menu.

A text report on the results of OptiLayer Monitor mode can be created using the Monitoring Report command from the Results menu.

The universal output command is Tabulate… in the Results menu, which allows the user to generate text output for virtually any design characteristic (e.g., transmission, reflection, absorption, phase, etc.).

In all cases, the text output is displayed on the screen.

There are three types of outputs that can be easily selected with the help of a special switch on the toolbar of the report window. They include Hierarchical view, Plain view, and Editable view.

  • Hierarchical view allows one to expand and collapse different topics of a report. The report can be printed as a whole, or only expanded topics can be selected.

  • Plain view displays all output report text in a usual way.

  • Editable view is similar to Plain view but allows editing of the output report by adding remarks, etc. You can edit it in the Output window that is opened once a report is generated. To store output as an ASCII file, use the Save As… command from the File menu. The user can print the report file using the Print… command from the File menu.

When a report is displayed, you can also access the Save As… and Print… commands from the popup menu invoked by the right mouse button. This popup menu also contains the most frequently used editing commands (Copy, Paste, etc.)

Any report can be easily exported to MS Excel using the corresponding button on the Output window toolbar or the Export to Excel command in the File menu.
