Spreadsheet Editing Tools

OptiLayer provides a powerful set of editing options that facilitate the entering and editing of data. Some of the key features in this set include:

  1. Insert options:

Users can easily insert new data points, layers, or parameters into the design files, allowing for adjustments and modifications to be made seamlessly. 2. Column editors:

Column editors enable users to edit specific columns of data, making it simple to modify individual parameters within the design files. 3. Grid generators:

Grid generators assist in creating and adjusting grids within the design files, aiding in visualization and organization of the data. 4. Drag-and-drop functionality:

Users can conveniently rearrange and reorder data points or layers by dragging and dropping them within the interface. 5. Copy and paste options:

Copying and pasting data within the software makes it easy to replicate information or transfer data between different sections of the design files. 6. Undo and redo capabilities:

Users can undo and redo editing actions, providing flexibility and the ability to revert changes if needed.

OptiLayer, all data editors include instant preview features that allow users to examine and validate various aspects of the design, such as refractive index spectral dependencies, spectral targets, and refractive index profiles. To edit spreadsheets effectively, OptiLayer provides a set of hotkeys and special commands accessible from the Edit Menu (some are available in the toolbar as well).

Here are some key functionalities and tips for editing data in OptiLayer:

  • To select a desired field, users can use the mouse pointer or the Arrow keys, PgUp, and PgDn keys.

  • Pressing the Enter key confirms a displayed value, while Ctrl+Enter is used as a general confirmation key for entering data in a spreadsheet.

  • When editing the last value in a spreadsheet, pressing Enter will confirm all spreadsheet data. Alternatively, users can press Ctrl+Enter (or click the OK button) to confirm all entries at once.

  • OptiLayer checks values for correct syntax and range. The Enter key checks the highlighted value locally, while the Ctrl+Enter key checks all data in the spreadsheet for validity.

These features and functionalities in OptiLayer streamline the editing process, ensure data accuracy, and provide a user-friendly interface for designing and optimizing coatings.

F-keys for editing spreadsheets:

In OptiLayer, there are various hotkeys and special commands that facilitate data editing and manipulation. Here are some additional key functionalities for efficient data editing within the software:

  • F3 - Inserts a new row below the active (highlighted) one.

  • F4 - Inserts a new row above the active one.

  • F6 - Invokes the Column Editor, which is used to edit data in columns by entering a starting value and an increment to define sequential values for the column.

  • F7 - Marks the beginning and end of a field to be selected or highlights a selected area. Pressing the key again while an area is highlighted will undo the selection.

  • F8 - Deletes a row identified by the cursor or an entire highlighted area.

  • F9 - Activates the special import file feature to import numerical data from an external text file - Import Numerical Data.

  • F10 - Displays data in the Paste buffer, allowing users to copy this data to the spreadsheet being edited.

  • Grid Generator (Shift+F4, Shift+F5, Shift+F6 keys): Enables users to fill spreadsheet columns with uniformly, logarithmically increasing or decreasing, or inversely proportional values.
