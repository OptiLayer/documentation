Hardware Requirements

` <optilayer_development.html>`__ ` <idh_optilayer.html>`__ ` <oc_installation.html>`__

OptiLayer can be used on any PC running Windows 10, Windows 8.1, Windows 7 (Service Pack 1), or Windows Vista (Service Pack 2 recommended). You have the option to use a 32-bit OS (in which case you must use the 32-bit edition of OptiLayer software) or a 64-bit OS (allowing you to use either the 32 or 64-bit editions of OptiLayer). The 64-bit edition of OptiLayer can utilize almost unlimited memory accessible to 64-bit systems, and the computational speed between the 32 and 64-bit editions is very similar. Older systems such as Windows 95, 98, ME, Windows NT 4.0, Windows 2000, and Windows XP are not supported and should be upgraded to a newer OS first. For optimal performance, it is recommended to use modern fast computers with a CPU speed of >1 - 2 GHz, at least 4GB of RAM, and a video resolution of 1024x768 or higher.

Important note: SSE3 instruction set should be supported by the CPU. See the full list of supported CPU at Wikipedia: SSE3 and supported processors. AVX2 vector instructions will be used automatically when available. CPU with AVX2 instruction support will allow to run computations significantly faster.

In the case of multi-processor systems or systems with multi-core processors additional improvement can be achieved, because OptiLayer can utilize more that one processor or processor core simultaneously during computations (parallel computations can be enabled in the Configuration menu using Multiprocessing option).

You can also use any of 64 bit systems (Windows XP x64, Windows Vista x64, Windows 7, Windows 8, 8.1 x64).

Server versions of OS (Windows Server 2008 R2 and Windows Server 2012 R2, Windows Server 2016) are also supported. In order to use Remote Connections (Terminal Services) it is necessary to purchase additional Server license of OptiLayer. The default license allows to use OptiLayer software either at the console or using single remote connection (this constitutes one working place)

In any case we recommend to install the latest upgrades or service packs for any OS used. Please, visit http://www.microsoft.com in order to check the availability of the latest updates for your OS or use Automatic Update function available in the last editions of Windows OS.
