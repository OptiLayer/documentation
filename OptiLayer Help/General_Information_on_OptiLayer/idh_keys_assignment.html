Keyboard Keys Assignment

OptiLayer Keys Assignments

Certain keystrokes have been assigned to activate the most commonly used OptiLayer features. This provides additional access to these features for user convenience. In the following hot keys list, menu commands with the same functionality are mentioned in parentheses:

File and Help system:

F1 (Help)

key is used to activate a Help topic for any dialog where a Help button may not be present.

Ctrl+O (Open Project File…)

This key is used  to open a Project file storing information on data loaded to memory, settings, and window layout.

Ctrl+Alt+Z (ZEMAX Coating)

Export current coating in ZEMAX Coating or ZEMAX Performance formats

Ctrl+Alt+O (OPTIS Coated Surface)

Export coating performance in OPTIS Coated Surface format

Ctrl+Alt+V (CODE V Sequence)

Export current coating in CODE V Sequence file format

Ctrl+Alt+F (FRED file)

Export current coating in FRED format

Ctrl+Alt+T (OpTaliX Multilayer)

Export current coating in OpTaliX Multilayer format

Editing tools access:

In OptiLayer, the following keys provide access to editing tools and are available when any database file is open for editing:

Ctrl+Z (Undo),

Ctrl+X (Cut),

Ctrl+C (Copy),

Ctrl+V (Paste)

Ctrl+A (Select All)

are standard Windows assignments

In OptiLayer, the following keys allow accessing special Spreadsheet Editing Tools:

F3 (Insert Above)

allows to insert a copy of marked row(s) before the current line

F4 (Insert Below)

allows to insert a copy of marked row(s) after the current line

F5 (Angles Editor)

starts a dialog to add or modify the number of incidence angles and their values

F6 (Column Editor)

starts a tool for easy filling spreadsheet columns

F7 (Mark Rows)

allows marking rows in a spreadsheet for subsequent editing operations

F8 (Delete Row)

deletes marked row(s) or currently selected row of a spreadsheet

F9 (Import)

is used to import data from external text files

F10 (Paste Imported)

starts a dialog to review imported data and to copy them to the data file opened for editing

Shift+Ctrl+F9 (Import XY data)

is used to import measurement data in 2-column text format

Keys to access OptiLayer View commands:

Ctrl+I (General Information)

opens General Information window with the information about current OptiLayer settings and loaded data

Alt+0 (Alt+zero) (Window List)

opens Window List window allowing easy switching between different OptiLayer windows

Keys to access commands in the Analysis menu:

Ctrl+E (Evaluation)

starts the Evaluation procedure for T & R & Phase

Ctrl+Alt+G (GD and GDD)

starts the Evaluation procedure for calculating and analyzing Group Delay and Group Delay dispersion characteristics

Ctrl+S (Ellipsometric Angles)

starts the Evaluation procedure for calculating and analyzing Ellipsometric Angles

Ctrl+F (Field)

plots the Electric Field distribution for the loaded design

Ctrl+Alt+A (Admittance)

plots Admittance diagram circles for the loaded design in a phase plane

Ctrl+D (Index Profile)

plots the refractive index profile of the loaded design

Ctrl+Alt+W (WDM Filter GD performance…)

open specialized window allowing to estimate phase properties of a WDM Filter with specified shape of transmittance characteristic

Ctrl+P (Plot Engine)

starts the OptiLayer Plot Engine, OptiLayer powerful graphics tool

Ctrl+Alt+E (Error Analysis)

starts the Errors analysis procedure

Ctrl+Alt+Y (Error Yield Analysis)

starts the Error Yield Analysis procedure

Ctrl+Alt+W (Worst Case)

starts the Worst Case analysis procedure

Ctrl+L (Layer Sensitivity)

starts the procedure ranging coating layers by their sensitivity to thickness errors

Keys to access commands in the Synthesis menu:

Ctrl+R (Refinement)

starts Refinement procedure

Ctrl+N (Needle Optimization)

runs Needle Optimization in the manual mode

Ctrl+U (Needle Optimization AUTO)

runs Needle Optimization in the automatic mode

Ctrl+G (Gradual Evolution)

starts the process of Gradual Evolution

Ctrl+W (WDM Filter)

starts special procedure for designing WDM filters

Ctrl+H (Inhomogeneous Refinement)

starts Inhomogeneous Refinement procedure

Ctrl+M (Random Optimization)

generates a set of Random Designs and optimizes these designs

Ctrl+Alt+D (De-Sensitization)

starts specialized de-sensitization option of OptiLayer aimed at obtaining coatings with decreased sensitivity to thickness errors

Ctrl+Q (Design Cleaner)

starts an automatic procedure for layer removal and design simplification (see Design Cleaner)

Ctrl+T (Thin Layer Removal)

allows the user to Remove Thin Layers

OptiMonitor access:

Shift+F3 (Monitor)

starts the OptiLayer Monitor routine generating the monitoring instructions for optical monitors

The following keys are of general use:

Enter key: Confirms a current operation or choice, usually equivalent to pressing the OK button.

Arrow keys: Allow navigation through the menu and between various edit fields. You can also select fields with the mouse pointer.

Esc key: Cancels a current operation, closes an edit window or a submenu without making any changes. Usually has the same meaning as pressing the Cancel button.

Ctrl+Enter key: Used as a general confirmation key, useful when entering data in a spreadsheet. Its action is the same as pressing the OK button.

Alt+Enter key: Used as a shortcut to the Apply button at the forms where this button exists.
