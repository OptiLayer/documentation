Foreword

To perform tutorial exercises you will need example directories installed at your hard disk.

If you are using retail version of OptiLayer, make sure that you selected “Custom” setup type during the installation, and that “Examples” topic at the next page of the setup procedure was selected.

Tutorial Exercises and Advanced Examples don’t require significant computational power. It is possible to use slow computers in order to perform all exercises.

At the welcome dialog enter your name and if you don’t want this dialog to be displayed each time you start OptiLayer you can disable it. If necessary it can be enabled again in the Configuration dialog.

OptiLayer uses decentralized data storage. In other words, data for each specific problem are located at a separate directory, named Problem directory.

You can open a Problem directory using File->Problem menu command. In order to use data in the OptiLayer program you will need to load all necessary data files from the disk using Load button of the Database dialog (for more details see Exercises Help).

Important Note: Please, don’t use Tutorial and Example directories for your own data. If you reinstall OptiLayer Thin Film Software, Tutorial and Example directories will be overwritten and any new data stored there will be lost.
