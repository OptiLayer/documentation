General Configuration

Configuration Options allow the user to set some of the most general settings that are used throughout the program. They include the following items:

  • Mode: Allows the selection of angular or spectral mode for spectrophotometric and ellipsometric measurements independently.

  • Units: Enables the user to select units for measuring lengths, frequencies, etc.

  • Thickness Units: Allows switching between physical, optical, quarter-wave, or full-wave representations of layer thicknesses.

  • Source/Detector: Enables the user to specify how light sources and detectors are utilized during computations of transmittance (T), reflectance (R), and intensity (I), as well as the units for spectral distribution.

  • Color: Provides options to set various parameters affecting the color evaluation module, including Reference White, Observer, and Integration step. Users can also specify a custom Reference White light source as a blackbody or from a Light Source/Detector database file.

  • GD & GDD: Allows the selection of units for Group Delay (GD), Group Delay Dispersion (GDD), and Third Derivative Dispersion.

  • Data: Offers adjustments to the appearance of the Database dialog and sets miscellaneous options, including sorting order for database files by Name, Author, Creation, or Revision Date.
