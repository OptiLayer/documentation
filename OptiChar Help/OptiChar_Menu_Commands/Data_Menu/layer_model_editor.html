Layer Model Editor

` <wavel ength_region_for_saved_da.htm>`__ ` <idh_menu_data.htm>`__ ` < idh_characterization_menu.htm>`__

This dialog is designed to edit layer models. The dialog caption shows the name of the data set being edited.

Layer model can be edited on disk with the help of Layer Model Editor. At the left side of the Editor, different parts of the current model can be selected in the tree view control. At the right part of the window, the corresponding parameters can be edited. The resulting model can be previewed while editing session.
Use Preview tab in order to see the current model parameters in graphical representation. Refractive index wavelength dependence, extinction coefficient wavelength dependence, and inhomogeneity factor can be plotted at the Preview window.
Press OK button to save edited model to disk. The Cancel button will discard modified values.
