Layer Material Database:

  • This database is utilized to store layer materials data.

  • For calculating with layer materials, the user must first Load them into computer memory (refer to Loading Data for details).

  • To access a database file, open the corresponding database access window by selecting the appropriate item from the Data menu or the OptiLayer Toolbar.

  • For information on how to edit materials properties, please refer to Editing Material Properties.
