Current Layer

` <idh_current_substrate.htm>`__ ` <idh_menu_view.htm>`__ ` <idh_substrate_plots.htm>`__
Opens the Current Layer window. This window displays the information about the optical parameters of the film that have been found during the layer characterization process.

Note: Data in this window cannot be edited.
