The Export to Excel Options dialog allows you to customize the format of the worksheet created in MS Excel.

Include Spreadsheet Header check box allows you to add header information to the Excel worksheet.
Export Only Selected Area option enables you to limit exporting to the selected region only.
Press the OK button to start the export procedure.
