Catalog

The current version of OptiChar features four catalogs: Light Source, Detector, Substrate, and Layer Material.

The Light Source catalog includes data on commonly used light sources in colorimetry.

The Detector catalog provides information on the spectral luminous efficiencies of the human eye.

The Layer Material catalog contains data on popular materials utilized in coatings production.

The Substrate catalog includes several sub-catalogs:

  • Schott Glass Catalog

  • Hoya Catalog

  • Ohara Catalog

  • Ohara Low Tg Catalog

  • Sumita Catalog

  • Hikari Catalog

  • CDGM Catalog

  • Metals Data: Information on various metals.

  • Miscellaneous Data: Files that do not belong to any other sub-catalogs.

To select a sub-catalog, use the Database combo box located at the top of the Catalog dialog, as illustrated below. Once you change the selection in this combo box, the catalog will display only the corresponding subset of data files.

The Find command activates the search feature, simplifying the selection of files based on a specified search pattern.

You can transfer many Catalog files at once. To do this, select multiple files using your mouse (hold the Ctrl and Shift keys for various multi-select options, similar to other Windows programs). After selecting the files in the Catalog dialog, press the Transfer button to complete the transfer operation.

Selected files will be placed in the current Problem Directory .
