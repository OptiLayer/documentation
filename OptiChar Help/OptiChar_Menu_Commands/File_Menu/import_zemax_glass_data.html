Import ZEMAX Glass data

Navigation:  OptiChar Menu Commands > File Menu >

` <refr activeindex_info_database.htm>`__ ` <idh_menu_file.htm>`__ ` <page_setup.htm>`__

ZEMAX files with the .AGF extension contain a set of dispersive data for various types of glass. The list of glasses available in the .AGF file is presented in the corresponding section. It is possible to select multiple items for import.

After pressing Import button selected items will be imported to the current Substrate database.

Note. *.AGF files may utilize dispersive formulas that are currently not supported by OptiLayer software. In such cases, the dispersive data will be automatically converted to table format. The wavelength grid will be selected to match the wavelength grid used by the Internal Transmittance data in the same file.
