Problem directory backup

Storing problem directory data can be quite handy for future use, transferring to another computer, or backing up to external media such as a floppy disk. OptiChar provides a feature that allows you to create an archived backup copy of any problem directory. Once you’ve created this archive, it can be used to restore your saved data whenever necessary. This ensures that your work is secure and accessible, protecting against data loss or facilitating easy data transfer.

You can back up one problem directory at a time. To perform a backup, select a problem directory from the list labeled “OptiLayer Problem Directories.” If needed, you can choose the OptiLayer Root Directory by pressing the Browse button or by selecting the name of a root directory from the combo box at the bottom of the list.

In the list box Backup on the right side of the dialog window, select the destination disk drive and (if necessary) the directory where a backup file will be created. OptiChar will generate a backup file with the name of the directory you are saving, followed by the “.zip” extension.

To complete the backup, press the Backup button. In the fields labeled From and To, located just below the button, you can see the names of the source directory and the destination file.

To restore data that has been previously saved, select the appropriate drive and (if necessary) the directory in the Backup frame. In the list of files at the bottom of the frame, all archive (“.zip”) files are listed. Select the file you need to restore.

OptiChar creates or overwrites a problem directory with the same name as that of the backup file.

To restore a saved problem directory, press the button Restore. In the fields From and To right below the button you can see the names of the source backup file and destination problem directory.

The radio button To Process allows you to include or exclude files other than OptiChar data files (for example, any documentation files that you may have in your problem directory). If you want to back up only OptiChar data files, select the Only Data Files option. Otherwise, select All Files.

Convert: to ver. 8.18 format option allows you to convert all data files to a format compatible with version 8.18 of OptiLayer Thin Film Software. Please note that settings related to new features not available in version 8.18 will be lost during this conversion.

Use Password to encrypt ZIP files created with the Backup operation. Specify the password in the related entry field.
