Horiba Ellipsometers

` <idh_woolam_options.htm>`__ ` <import_menu.htm>`__ ` <sopra_ellipsometers.htm>`__

Horiba ellipsometers are able to save measured ellipsometric angles to files with the extension *.spe.

OptiChar will:

  • open the file

  • convert it to a Measurement file automatically.

Since information on wavelength units is stored inside Horiba files, no additional options should be specified.
