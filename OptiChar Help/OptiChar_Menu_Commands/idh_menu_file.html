File Menu

` <quickstart.htm>`__ ` <idh_optilayer.htm>`__ ` <idh_problem_directory.htm>`__

The File submenu in OptiChar is designed to enhance data management, particularly in Decentralized Storage Mode (DS Mode). Here are the main features:

  • Access to Problem Directories:

    • Gain access to various problem directories that are stored directly on your hard disk.

  • Transfer Data Files:

    • Easily transfer data files between different problem directories, streamlining your workflow.

  • Backup and Restore:

    • Utilize backup options to safeguard your directories and restore them whenever necessary, ensuring your data is secure.


It has options for saving and printing results along with the capability to export and import data in various formats. The File menu includes:

  • **Problem - **Allows the user to select a / Problem Directory/ .

  • **Recent Problems - **Facilitates quick selection of previously opened Problem Directories.

  • File Transfer - Opens a submenu to initiate the / File Transfer/ utility for exchanging data between different Problem Directories.

  • Backup/Restore - Opens a dialog to create a backup of your Problem Directory or to restore previously saved data (refer to / Problem Directory Backup/ ).

  • Catalog - Enables loading substrates from a / catalog/ of known substrates. Currently, this includes Shott, Hoya, Ohara, Ohara Low Tg, Metals, CDGM, Sumita, Hilkari, and other data files. Information on Layer Materials is also available. Additionally, catalogs for Light Sources and Detectors are included.

  • Refractiveindex.INFO - Opens a database available online at / http://refractiveindex.info/ , which allows users to select information on refractive indices and extinction coefficients of various materials for import into OptiChar for further applications.

  • OpenLayout - Opens a previously saved OptiChar Layout file (.ocprj extension) and updates all settings related to the loaded data, configuration, and window layout based on the information stored in the OptiChar Layout file. This option is also accessible via the Ctrl+O keyboard shortcut.

  • SaveLayout As - Allows the user to save an OptiChar Layout file (.ocprj file extension) for future use.

  • Save Chart - Opens a standard File Save dialog box to specify the name of the output file. If text output of any data is generated and the text output window is active, an ASCII file with the default extension .out is created (see / Creating Text Output/ ). If the current window is to be saved as an image, options for various graphic formats are available (.bmp, .jpg, .wmf, .gif, .png, .pcx, .tiff, etc.). Additionally, if a chart was selected prior to this operation, the .vtc2 format supported by the OptiLayer Plot Engine will also be available.

Import

  • Sopra n, k Data - Allows for the import of Sopra dispersion data directly into OptiLayer. The Sopra database can currently be downloaded freely from / http://www.sopra-sa.com/ . Imported files can be placed into either the Substrate/Incident Medium database or the Layer Material database. The database type is determined by the appropriate selection in the “Files of Type” field within the Import Sopra n,k Data dialog.

  • ZEMAX Glass Data- Facilitates the import of dispersion data for a material or substrate represented in the ZEMAX Glass format (.agf). Upon opening the .agf file, the / Import ZEMAX Glass Data/ dialog will allow users to select the materials to be imported.

  • Bühler Laybold Disprsion Data - Enables the import of dispersion data for Substrates, Incident Medium, and Layer Materials stored in the / Bühler Leybold format/ (*.ldd files).

Export

  • / Export to Excel… (F11) - Allows the User to Export Any Spreadsheet or Report (see / Results Menu/ ) to Microsoft Excel

  • Export Chart - Opens the / Export Chart/ dialog, enabling you to save the currently selected chart in various graphical or data formats.


  • Page Setup - Opens the / Page Setup/ dialog for configuring page layout and settings.

  • Print Preview - Opens the / Print Preview/ dialog, allowing users to view and adjust the printer output. The Print Preview feature is available only for windows that display plots.

  • Print - Enables printing of a text output directly to the printer instead of saving it to a file. This option is also available when generating text output (see the section on / Creating Text Output/ ).

  • Exit - Quits the program. This option can also be accessed using the keyboard shortcut Alt-X
