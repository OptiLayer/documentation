Hardware Requirements

` <optilayer_development.htm>`__ ` <idh_optilayer.htm>`__ ` <oc_installation.htm>`__

OptiChar can be used on any PC running Windows 10, Windows 8.1, Windows 7 (Service Pack 1), or Windows Vista (Service Pack 2 is recommended). You can utilize a 32-bit operating system (in which case you must use the 32-bit edition of OptiLayer software) or a 64-bit operating system (in which case you can use any edition of OptiLayer: either 32-bit or 64-bit). The 64-bit edition of OptiLayer can access nearly unlimited memory available to 64-bit systems, and the computational speed of the 32-bit and 64-bit editions is very similar.

Old systems (Windows 95, 98, ME, Windows NT 4.0, Windows 2000, and Windows XP) are NOT supported and should be upgraded to a newer operating system first.

For optimal performance, it is recommended to use modern, high-speed computers with a CPU speed of greater than 1-2 GHz, at least 4 GB of RAM, and a video resolution of 1024x768 or higher.

Important note: Important note: The CPU must support the SSE3 instruction set. You can find the full list of supported CPUs on Wikipedia under SSE3 and supported processors. Additionally, AVX2 vector instructions will be utilized automatically when available. CPUs that support AVX2 instructions will enable significantly faster computation performance.

In the case of multi-processor systems or systems with multi-core processors additional improvement can be achieved, because OptiChar can utilize more than one processor or processor core simultaneously during computations (parallel computations can be enabled in the Configuration menu using the Multiprocessing option). Server versions of OS (Windows Server 2008 R2, Windows Server 2012 R2, Windows Server 2016) are also supported. In order to use Remote Connections (Terminal Services) it is necessary to purchase an additional Server license of OptiChar. The default license allows using OptiLayer software either at the console or using a single remote connection (this constitutes one working place). In any case, we recommend installing the latest upgrades or service packs for any OS used. Please visit ` <http://www.microsoft.com/>`__http://www.microsoft.com to check the availability of the latest updates for your OS or use the Automatic Update function available in the latest editions of Windows OS.
