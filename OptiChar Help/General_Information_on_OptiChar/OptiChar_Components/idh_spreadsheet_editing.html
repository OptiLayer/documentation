Spreadsheet Editing

The system provides a powerful suite of editing options to facilitate the entry and modification of data. Below is an overview of the key features available:

  1. Insert Options: Users can easily insert new rows, columns, or data entries into the existing database. This functionality allows for efficient data management and customization.

  2. Column Editors: Specialized tools for editing entire columns within the data grid. These editors enable users to modify data types, formats, and apply bulk changes across selected columns, streamlining the editing process.

  3. Grid Generators: This feature allows users to generate data grids populated with predefined or user-defined data sets. Grid generators help in quickly creating the necessary data structure without manual entry.

  4. And Much More: Additional editing functionalities may include features such as:

  • Data validation tools to ensure accuracy and consistency.

  • Copy and paste capabilities.

  • Search and filtering options for easy navigation through data.

  • Undo and redo functionalities for error correction.

  • Import and export capabilities for data integration with other applications or formats.

OptiChar offers an impressive array of functionalities for data editing with instant preview features. This allows users to effectively assess and validate various parameters such as:

  • Refractive Index Spectral Dependencies

  • Spectral Targets

  • Refractive Index Profiles of Various Designs

To enhance the user experience while editing spreadsheets, OptiChar provides several tools including hotkeys and special commands from the Edit Menu, some of which are also accessible via the Toolbar.

  1. Selecting Fields:

    • Utilize the mouse pointer or the Arrow keys to navigate through fields.

    • The PgUp and PgDn keys can also be employed for quicker navigation.

  2. Confirming Values:

    • Pressing the Enter key confirms the displayed value in the current field.

    • If you are editing the last value in a spreadsheet, pressing Enter will confirm all the data in the spreadsheet at once.

  3. General Confirmation:

    • Use Ctrl+Enter as a general confirmation key, which efficiently confirms all entries in a spreadsheet without needing to confirm them one by one.

    • Alternatively, the OK button can also be used for confirming all entries simultaneously.

OptiChar ensures data integrity through built-in syntax and range checks:

  1. Local Validation: The Enter key checks the highlighted value for correct syntax and range locally.

  2. Spreadsheet Validation: The Ctrl+Enter key validates all entries in the spreadsheet, ensuring that all data adheres to expected formats and values.


F-keys for Editing Spreadsheets in OptiChar

their functionalities:

  • F3 - Inserts a new row below the active (highlighted) one.

  • F4 - Inserts a new row above the active one.

  • F6 - Invokes the Column Editor for editing data in columns. Upon pressing F6, a window will prompt you to enter a starting value and an increment. After confirming with the Enter key, the specified values will fill the entire column or the highlighted part of it.

  • F7 - The first press selects the beginning of a field, while the second press marks the end and highlights the selected area. Pressing it a third time will undo the entire selection.

  • F8 - Deletes the row identified by the cursor or an entire highlighted area if one exists.

  • F9 - Activates a special import feature to import numerical data from an external text file. A prompt will appear for you to enter the name of the desired file, and upon selection, the file’s contents will display in a new window.

  • F10 - Displays data from the Paste buffer, allowing you to copy this data into the spreadsheet you are editing.
