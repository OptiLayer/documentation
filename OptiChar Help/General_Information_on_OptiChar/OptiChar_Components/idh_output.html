Output Options

Results obtained through OptiChar can be presented in multiple report formats to cater to diverse user needs. Here’s a detailed overview of the reporting capabilities and export options available:


Types of Reports

  • Design Report

  • Evaluation Report

  • Errors Report

  • Layer Material Report

  • Other Custom Reports


Report Formats

Reports can come in various formats to meet sophisticated user requirements, including:

  • Hierarchical Form: Allows for expanding or collapsing different topics.

  • Editable Form: Users can add remarks and modify the content.

  • Plain Form: Displays report text in a straightforward format.


Copying Reports

Users can select any report area and easily copy it to the Windows clipboard for quick sharing or pasting into other applications.


Saving and Printing Options

Users have the flexibility to save and print any report or OptiChar window in multiple formats:

1. Image Formats:

  • *.bmp (bitmap graphics format)

  • *.jpg (JPEG bitmap)

  • *.wmf (Windows metafiles)

  • *.pcx (Paintbrush format)

  • *.gif (CompuServe bitmap)

  • *.png (Portable Networks Graphics)

  • *.tiff (Tagged Image File Format)

2. Chart Windows Export Formats:

  • *.vtc2 (used by OptiLayer Plot Engine II)

  • *.pdf (for opening with Adobe Acrobat tools, http://www.adobe.com)

  • *.eps (Encapsulated PostScript files)

  • *.svg (Scalable Vector Graphics, http://www.w3.org/Graphics/SVG)

  • *.vml (Vector Markup Language, http://www.w3.org/TR/NOTE-VML)

  • *.xaml (Extensible Application Markup Language, http://en.wikipedia.org/wiki/Extensible_Application_Markup_Language)

Note: OLE (Object Linking and Embedding) is supported by the Plot Engine II.

Direct Export to MS Excel

OptiChar simplifies the export process by allowing users to export reports directly to MS Excel with just one click.


Generating Text Reports

OptiChar can generate text outputs for:

  • Database file content: Click the Output button in the Database Window menubar.

  • Fitting details of measurement data: Use the Fitting Report command from the Results menu.

  • Current Substrate and Layer model: Select the relevant command from the Results menu.

Upon generation, the text output is displayed on the screen.


Output Views

Users can select from three types of output views using a special switch in the report window:

  1. Hierarchical View: Expand/collapse topics; option to print the entire report or selected topics.

  2. Plain View: Standard display of all report text.

  3. Editable View: Similar to Plain view but allows editing for adding remarks, etc.


Saving and Printing Reports

  • To save output as an ASCII file, use the Save As… command from the File menu.

  • To print a report, users can utilize the Print… command from the same menu, or via the right-click popup menu, which also offers common editing commands like Copy and Paste.


Exporting Reports

Any report can be exported to MS Excel conveniently using the dedicated button in the Output window toolbar or through the Export to Excel command in the File menu. By providing versatile reporting formats and export capabilities, OptiChar meets the varied requirements of its users effectively.
