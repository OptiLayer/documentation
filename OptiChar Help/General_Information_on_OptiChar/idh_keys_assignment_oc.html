Keyboard Keys Assignment

Here’s a comprehensive list of keyboard shortcuts (hotkeys) for commonly used features in OptiChar. These shortcuts enhance user convenience by providing quick access to various functions within the software.


File and Help System Hotkeys:

  • F1 (Help): Activates a Help topic for any dialog where a Help button may not be present.

  • Ctrl + O (Open Project File…): Opens the project file storing information on data loaded to memory, settings, and windows layout.


Editing Tools Access:

When any database file is open for editing, the following standard editing tools shortcuts are available:

  • Ctrl + Z: Undo

  • Ctrl + X: Cut

  • Ctrl + C: Copy

  • Ctrl + V: Paste

  • Ctrl + A: Select All


Special Spreadsheet Editing Tools

  • F3: Insert Above (Inserts a copy of marked row(s) before the current line)

  • F4: Insert Below (Inserts a copy of marked row(s) after the current line)

  • F5: Angles Editor (Starts a dialog to add or modify incidence angles and their values)

  • F6: Column Editor (Starts a tool for easy filling of spreadsheet columns)

  • F7: Mark Rows (Allows marking of rows in a spreadsheet for subsequent editing operations)

  • F8: Delete Row (Deletes marked row(s) or currently selected row in the spreadsheet)

  • F9: Import (Used to import data from external text files)

  • F10: Paste Imported (Starts a dialog to review imported data and copy it to the open data file)


OptiChar View Commands:

  • Ctrl + I: General Information (Opens the General Information window with details about current settings and loaded data)

  • Alt + 0: Window List (Opens the Window List window for easy switching between different OptiChar windows)

  • Ctrl + F: Data Fitting (Plots Data Fitting of measurement data)

  • Ctrl + P: Partial Discrepancies (Plots Partial Discrepancies of measurement data)

  • Ctrl + U: Substrate Plots (Opens Substrate Plots window with refractive index and extinction coefficients of the currently loaded substrate)

  • Ctrl + Y: Layer Plots (Opens Layer Plots window with the refractive index, extinction coefficient, and inhomogeneity factor of the layer)


Import Options:

  • Shift + Ctrl + F5: Multiple Files (Allows importing many files in one operation with autodetection of file format)

  • Shift + Ctrl + F6: Import Spectra from Excel (Automated recognition of data positions)

  • Shift + Ctrl + F9: Import XY Data (Imports measurement data in a 2-column text format)

  • Ctrl + F11: Import Agilent Cary (Imports spectral photometric data in Agilent Cary format)

  • Ctrl + F9: Import JCAMP-DX (Imports measurement data in the JCAMP-DX format)

  • Ctrl + F12: Import Perkin Elmer (Imports spectral photometric data in binary Perkin Elmer format)

  • Shift + Ctrl + F12: Import Hitachi (Imports spectral photometric data in binary Hitachi format)

  • Shift + F12: Import JASCO (Imports spectral photometric data in binary JASCO format)

  • Ctrl + F10: Import Woollam (Imports ellipsometric measurement data used by Woollam ellipsometers, including R, T, and depolarization data)

  • Shift + Ctrl + F10: Import Horiba (Imports Horiba ellipsometric data)

  • Shift + F10: Import Semilab - Sopra (Imports Semilab - Sopra ellipsometric data)

  • Shift + Ctrl + F11: Import Sentech (Imports Sentech ellipsometric data)


OptiChar Characterization Tools Hotkeys:

  • Ctrl + L: Starts Layer Characterization procedure.

  • Ctrl + R: Starts Layer Refinement procedure.

  • Ctrl + Alt + I (Index Shift): Starts Index Shift procedure.

  • Ctrl + Alt + I (Mixture): Starts Mixture Characterization procedure.

  • Ctrl + Alt + I (Porosity): Starts Porosity Characterization procedure.

  • Ctrl + B: Starts Bulk Inhomogeneity procedure.

  • Ctrl + Alt + A: Starts Surface Inhomogeneity procedure.

  • Ctrl + M: Opens Simplify Model dialog for removing certain features of the current layer model.

  • Ctrl + S: Starts Substrate Characterization procedure.


General Use Keys:

  • Enter: Confirms the current operation or choice, similar to pressing the OK button.

  • Arrow Keys: Navigate through menus and between various edit fields. You can also select fields with the mouse pointer.

  • Esc: Cancels the current operation, closes an edit window or a submenu without making any changes, equivalent to pressing the Cancel button.

  • Ctrl + Enter: A general confirmation key useful when entering data in a spreadsheet, acting as if you pressed the OK button.

  • Alt + Enter: Serves as a shortcut to the Apply button where applicable.
